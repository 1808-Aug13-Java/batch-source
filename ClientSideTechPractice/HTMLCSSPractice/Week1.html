    • Benefits of OOP

    • Automatic memory management
    • Incredibly widespread
    • Free
    • Compile-time and runtime safety
  • Java Pieces
    • JDK: Java Development Kit is a superset of the JRE and JVM. It has the compiler.
    • JRE: Java Runtime Environment is a superset of the JVM + libraries needed to run code.
    • JVM: Java Virtual Machine interprets compiled bytecode
    • Use javac command to compile
    • Use java command to run the executable

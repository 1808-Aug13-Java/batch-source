Headers and Footers

Related to divitis, a common problem is the use of divs with a class similar to "header". If you find yourself using divs with a class of header or footer, you should probably just use the semantic HTML tags header and footer. This will make your document structure more readable and easier to understand. Again, divs have no semantic value.
